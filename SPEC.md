# eFix — Выездной сервисный центр

**eFix** — сайт для выездного сервисного центра по ремонту цифровой техники. Клиент оставляет заявку, мастер забирает устройство, чинит в мастерской и возвращает обратно. Всё онлайн: заказ, прозрачные цены, карта партнёрских мастерских, трекинг.

---

## Клиентская часть

### Главная (/)
- **Шапка**: логотип, меню (Главная, Услуги, Цены, О нас, Контакты), телефон, кнопка "Заказать звонок" — навигация всегда видна при скролле
- **Hero-блок**: крупный заголовок, подзаголовок про услуги, две кнопки («Вызвать мастера», «Смотреть цены»), три счётчика (лет опыта, отремонтировано, довольных клиентов) + форма обратного звонка прямо на hero
- **Блок услуг**: 4 карточки-категории (Телефоны, Планшеты, Ноутбуки, ПК) с иконкой, названием, описанием и ценой «от» — кликабельные, ведут на страницу услуг
- **Преимущества**: 6 причин выбрать нас (забор за час, бесплатная диагностика, гарантия до года, прозрачные цены, сложность любая, срочный ремонт) — пронумерованы, с акцентным фоном
- **Процесс ремонта**: 5 шагов (Заявка → Забор → Диагностика → Ремонт → Доставка) — горизонтальные карточки
- **Отзывы**: 3 карточки с аватаром, именем, звёздами, текстом и датой
- **CTA-блок**: призыв оставить заявку с кнопкой
- **Подвал**: лого, описание, ссылки на услуги и страницы, контакты (телефон, почта, адрес), иконки соцсетей (WhatsApp, Telegram, VK, Instagram, YouTube — настраиваемые), копирайт
- **Плавающая кнопка** — заказать звонок (всегда видна в правом нижнем углу)

### Страница "Услуги" (/services)
- Заголовок + подзаголовок
- Сетка категорий услуг — карточки-плитки с иконкой, названием, описанием
- При клике на категорию — модальное окно с заголовком, иконкой, описанием категории и списком конкретных услуг (название + цена)
- Внизу модалки — кнопка «Оставить заявку на ремонт»
- CTA-секция снизу

### Страница "Цены" (/prices)
- Заголовок + подзаголовок
- Блок фильтров: тип устройства (все/телефоны/планшеты/ноутбуки/ПК), бренд, модель, поиск по тексту
- Фильтры динамические — при выборе типа подгружаются бренды, при выборе бренда — модели
- Результаты — таблица с колонками: тип, бренд, модель, услуга, цена
- Пагинация
- Примечание: «Диагностика — бесплатно!»
- CTA-секция

### Страница "О нас" (/about)
- Заголовок + подзаголовок
- Текст о компании
- Счётчики (лет работы, отремонтировано, отзывов, время забора)
- Ценности: 3 карточки с иконкой (честность, скорость, гарантия)
- Партнёрская сеть: статистика (точек приёма, районов, категорий)
- Карта: интерактивная карта с маркерами партнёрских мастерских (Leaflet, OSM)
- График работы
- CTA-секция

### Страница "Контакты" (/contacts)
- Заголовок + подзаголовок
- Две колонки:
  - Левая: контакты (телефон, почта, адрес с иконками) + соцсети
  - Правая: форма заявки (имя, телефон, тип устройства, модель, описание проблемы)
- Отправка формы через AJAX (без перезагрузки), показ успеха

### 404
- Страница «Страница не найдена» с ссылкой на главную

### Модальное окно "Заказать звонок"
- Открывается с любой страницы по клику на плавающую кнопку или кнопку «Заказать звонок» в шапке
- Форма: имя, телефон, тип устройства, модель, описание
- После отправки — сообщение об успехе
- Плавное появление/исчезновение

### Аналитика (фоновая)
- Каждый просмотр страницы сохраняется: путь, IP, user-agent, referrer, экран, язык, UTM-метки, session_id
- Отслеживание новых/вернувшихся посетителей
- Поисковые запросы сохраняются
- Отправка форм отслеживается
- IP-геолокация (страна, город, координаты)

## Админ-панель (/admin/)

### Логин (/admin/login)
- Форма входа: имя пользователя + пароль
- Если нет админов — редирект на регистрацию

### Дашборд (/admin/dashboard)
- Карточки: всего заявок, просмотров сегодня, уникальных сегодня, за неделю, за месяц, позиций в прайсе
- Мини-график просмотров за месяц (спарклайн)
- Последние визиты (путь, IP, время)
- Популярные страницы
- Быстрые ссылки на разделы

### Редактор контента (/admin/site)
- Список всех ключей контента (пара ключ-значение)
- Редактирование любого параметра: название сайта, телефон, email, адрес, часы работы, hero-тексты, CTA-тексты, статистика (цифры и подписи), SEO (meta title/description), соцсети (URL и вкл/выкл), футер, копирайт, согласие на обработку
- Удаление лишних ключей

### Услуги (/admin/services)
- CRUD: список услуг (таблица), добавление, редактирование, удаление
- Поля: название, описание, цена, иконка, категория (phones/tablets/laptops/pc), сортировка
- Каждая услуга отображается в модалке на странице услуг

### Прайс-лист (/admin/prices)
- CRUD: таблица с пагинацией, добавление, редактирование, удаление
- Поля: тип устройства, бренд, модель, услуга, цена от/до, активно/неактивно
- Фильтрация по типу устройства

### Заявки (/admin/requests)
- Список заявок: имя, телефон, устройство, модель, сообщение, статус, дата
- Статусы: новая, в работе, готова, архив
- Смена статуса, добавление заметок администратора
- Массовые действия (изменить статус для выбранных)
- Бейдж новых заявок в сайдбаре с автообновлением каждые 30 сек

### Партнёрские мастерские (/admin/workshops)
- CRUD: список, добавление, редактирование, удаление
- Поля: название, адрес, координаты (lat/lng), телефон, описание, активно/неактивно
- Мастерские отображаются на карте на странице «О нас»

### Аналитика (/admin/stats)
- **Виджеты-счётчики** с анимацией: просмотров сегодня, уникальных сегодня, за неделю, за месяц, уникальных за месяц, заявок, всего просмотров, сессий
- **График просмотров по дням** (линейный, Chart.js, 14 дней)
- **Типы устройств** (пончик-диаграмма)
- **Карта географии** посетителей (Leaflet, маркеры с пульсацией)
- **Браузеры** (горизонтальные бары, grouped)
- **ОС** (горизонтальные бары, grouped)
- **Экраны** (таблица: разрешение → количество)
- **UTM-метки** (таблица: source, medium, campaign → клики)
- **Языки** (таблица: язык → количество)
- **Страницы** (таблица: путь → просмотры)
- **Referrers** (таблица: откуда → переходы)
- **Частые поиски** (таблица: запрос → раз)
- **Формы** (таблица: форма, действие → количество)
- **Активность сейчас** (список: путь, IP, время — за последние 15 минут, автообновление каждые 15 сек)
- Кнопка «Обновить» (все данные)
- Анимации появления элементов

### Настройки SMTP (/admin/mail-config)
- Форма: хост, порт, пользователь, пароль, TLS, from-email, from-name
- Уведомления: вкл/выкл, email для уведомлений
- Тестовая отправка письма

### Шаблоны писем (/admin/mail-templates)
- Список шаблонов
- Добавление/редактирование: name, subject, body
- Удаление

### Настройки (/admin/settings)
- Список системных настроек: ключ → значение
- Редактирование в один клик (через форму в строке)
- Редактор .env файла (прямое редактирование переменных окружения)

## Желаемый вид и стиль

- **Современный, минималистичный дизайн** — чисто, аккуратно, много воздуха
- **Цвета**: тёмно-синий (#0B2447) как основной, ярко-оранжевый (#FF6B35) как акцент — создаёт контраст и доверие
- **Шрифты**: Montserrat (заголовки) + Inter (текст) — современные, читаемые
- **Анимации**: плавный hover на карточках, fade-in при скролле, микроанимации на кнопках
- **Адаптивность**: мобильные, планшеты, десктоп — всё едет корректно
- **Карточки** с тенью и скруглениями (radius 12–16px)
- **Формы** на тёмном фоне hero — полупрозрачные, с blur-эффектом
- **Плавающая кнопка** — всегда видна, привлекает внимание
- **Интерактивная карта** (Leaflet + OSM) с кастомными маркерами
- **Графики и анимации в админке** — визуально приятные, данные понятны с первого взгляда
- **Сайдбар админки** — тёмный, с иконками, сворачивается, бейджи новых заявок
